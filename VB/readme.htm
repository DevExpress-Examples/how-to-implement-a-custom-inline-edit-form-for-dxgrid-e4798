Developer Express Example
How to implement custom dropdown row editor for DXGrid

This example demonstrates how to edit grid rows using a separate inline edit form with editors generated automatically based on the underlying row data object. To see this approach in action, click the "Edit" hyperlink in the "#" grid column. To close the edit form, click "Edit" again.
